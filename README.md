# Admission Follow-up CRM

College admission follow-up software for staff and managers.

## Download

👉 [Download Latest Installer](../../releases/latest)

## Features

- Staff PIN login (mobile)
- Manager dashboard (laptop)
- Auto WhatsApp messages (Tamil)
- AI photo scan for leads
- Fee tracking
- Analytics
- Auto reminders

## Install

1. Download `Admission CRM Setup.exe` from Releases
2. Run the installer
3. Software opens automatically
